# Codexify

Codexify is an RPI orchestration kit for Codex with reusable:
- role configs (`.codex/agents`)
- execution playbooks (`playbooks`)
- artifact templates (`templates`)
- versioned process contracts (`schemas`)

## Quick Install
Tell Codex:

```text
Open and follow instructions from .codex/INSTALL.md in this repository.
```

If this repo is hosted on GitHub, you can also use a raw URL to `.codex/INSTALL.md`.
